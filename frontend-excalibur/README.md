# Frontend Excalibur

Interfaz operativa para autenticacion y administracion de usuarios.

## Stack

- Vue 3
- TypeScript
- Vite
- Axios
- Lucide Vue

## Ejecutar

```bash
npm install
npm run dev
```

Por defecto usa el backend:

```text
http://192.168.1.94:8083
```

Se puede cambiar con:

```bash
VITE_EXCALIBUR_API_URL=http://host:puerto npm run dev
```
